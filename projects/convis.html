ConVis - Interactive Large-scale Neuron Network Visualization

Overview

ConVis: a web-based visualization system for large-scale connectome exploration that supports identity-preserving semantic zooming across organizational levels. The system enables seamless navigation between neuron populations and individual neurons while maintaining biological context, integrating hierarchical organization, 2D semantic overview, and 3D morphology visualization.

Features

  • Identity-preserving semantic zooming
  • Hierarchical organization visualization
  • 2D semantic overview
  • 3D morphology visualization
  • Seamless navigation between different organizational levels

Applications

ConVis is designed to help researchers explore and understand large-scale connectome data, making it easier to navigate complex neural networks and identify patterns in connectivity.

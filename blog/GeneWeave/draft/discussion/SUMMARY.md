# GeneWeave项目总结

## 项目概述

GeneWeave项目旨在理解基因如何决定神经系统的连接和形态，并开发从基因表达预测神经连接的计算模型。

## 核心发现总结

### 机制层面

**1. 轴突引导的三层机制**：
```
转录因子（身份决定）
    ↓
引导分子受体（方向感知）
    ↓
细胞骨架重组（物理运动）
```

**2. 突触特异性的分子代码**：
- Neurexin/Neuroligin：通用突触组装
- Beat/Side：层特异性（果蝇）
- DIP/Dpr：伙伴选择（果蝇）
- Protocadherin：自我回避（哺乳动物）

**3. 拓扑映射的梯度原理**：
```
投射位置 ∝ 受体表达量 / 配体浓度梯度
```

### 计算层面

**1. 双线性模型的有效性**：
- 少数基因对（~100对）可解释大部分连接
- 可解释性强，适合发现关键基因

**2. 深度学习的潜力**：
- 图神经网络可利用网络结构
- 但需要更多数据和更好的可解释性

**3. 符号回归的前景**：
- 可以发现简洁的数学规律
- 直接指导实验设计

### 实验层面

**1. 关键实验室的贡献**：
- Liqun Luo：排斥引导机制
- S. Lawrence Zipursky：Beat/Side系统
- Thomas Südhof：Neurexin/Neuroligin功能
- Albert-László Barabási：双线性建模

**2. 技术突破**：
- FlyWire：完整果蝇连接组
- MICrONS：哺乳动物连接组 + 功能
- 空间转录组：基因表达的空间分布
- CRISPR筛选：系统性基因功能鉴定

## 关键洞见

### 1. 遗传决定论 vs 随机性
- **C. elegans**：高度刻板，遗传决定为主
- **哺乳动物**：显著变异，活动依赖调整
- **平衡**：遗传提供框架，经验精细调整

### 2. 分子特异性 vs 空间约束
- **Peters规则**：某些细胞类型遵循（空间接近）
- **分子识别**：某些细胞类型需要特异性配对
- **整合**：空间预组织 + 分子精细化

### 3. 保守性 vs 物种特异性
- **保守**：Neurexin/Neuroligin, Eph/Ephrin, Semaphorin
- **特异**：DIP/Dpr（昆虫），Protocadherin簇（脊椎动物）
- **意义**：核心机制保守，细节适应性进化

### 4. 结构 vs 功能
- **结构连接 ≠ 功能连接**
- 需要整合电镜 + 功能成像
- 连接强度和可塑性同样重要

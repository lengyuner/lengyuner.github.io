## 核心问题与答案索引

### Q1: 基因如何决定轴突投射方向？
**答案位置**：文档1 - 轴突引导章节  
**关键机制**：转录因子 → 引导分子受体 → 生长锥感知 → 轴突转向  
**代表研究**：Liqun Luo实验室的ORN研究

### Q2: 神经元如何选择正确的突触伙伴？
**答案位置**：文档1 - 突触特异性章节  
**关键机制**：SAM配对（Neurexin-Neuroligin, Beat-Side等）  
**代表研究**：Zipursky实验室的T4/T5研究

### Q3: 如何形成有序的拓扑映射？
**答案位置**：文档1 - 梯度与排斥章节  
**关键机制**：Eph/Ephrin梯度 + 差异化受体表达  
**代表研究**：视网膜-视顶盖映射

### Q4: 如何从基因表达预测连接？
**答案位置**：文档2 - 双线性模型章节  
**关键方法**：P(i→j) = σ(g_i^T W g_j)  
**代表研究**：Barabási实验室的C. elegans工作

### Q5: 哪些基因对连接最重要？
**答案位置**：文档3 - 关键分子验证章节  
**关键基因**：Neurexin, Neuroligin, Beat, Side, Eph, Ephrin, Semaphorin  
**验证方法**：敲除、过表达、拯救实验

### Q6: 神经元形态如何发育？
**答案位置**：文档1 - 神经元形态章节  
**关键机制**：转录因子 → 细胞骨架调控 → 分支规则  
**代表研究**：Cuntz的最小布线模型

### Q7: 如何整合多组学数据？
**答案位置**：文档2 - 数据整合章节  
**关键策略**：早期融合、晚期融合、注意力融合  
**应用案例**：MICrONS项目

### Q8: 如何验证预测的连接？
**答案位置**：文档2 - 评估与验证章节，文档3 - 技术突破章节  
**验证方法**：电镜重建、光学追踪、功能成像  
**评估指标**：AUROC, AUPRC, 网络拓扑保真度

## 未来研究方向

### 1. 完整的分子图谱
- **目标**：系统性鉴定所有SAM及其配对规则
- **方法**：高通量结合实验 + AlphaFold预测
- **相关文档**：文档1, 3

### 2. 定量预测模型
- **目标**：从基因表达定量预测连接概率
- **方法**：整合多组学 + 深度学习
- **相关文档**：文档2

### 3. 跨物种比较
- **目标**：理解保守机制和进化创新
- **方法**：比较连接组学 + 进化基因组学
- **相关文档**：文档1, 3

### 4. 人工设计回路
- **目标**：通过改变基因表达重编程连接
- **方法**：合成生物学 + 基因编辑
- **相关文档**：文档1, 3

### 5. 疾病机制
- **目标**：理解连接异常与神经精神疾病
- **方法**：疾病模型 + 连接组分析
- **相关文档**：文档1, 3
